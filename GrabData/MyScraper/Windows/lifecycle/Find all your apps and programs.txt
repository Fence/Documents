Looking for a long lost app or program
The Start menu is the place to well start
Select the Start button and then scroll through the alphabetical list on the left
To limit scrolling jump to a specific part of the list
Select any letter in the alphabetical list and then choose the letter that the name of the app begins with
If you still can t find what you re looking for do a quick search for it
In the search box on the taskbar type the name of the app or program or press the Windows logo key on your keyboard and start typing
